Anchors (i.e. links)

Anchor tags create links to other pages. The URL goes in the 'href' attribute. Go to DuckDuckGo to search for more information.

Setting 'target="_blank"' will open the link in a new window. See MDN for more details.

Anchor tags can also create links to a new location on the same page. Instead of putting a URL in the "href" attribute, just use the "id" of the element you want to navigate to. This allow you to jump to the top of the page.

Text Emphasis

Italic

The 'em' element adds extra exphasis to the text within the opening and closing tags. This shows up as italic text in most browsers.

Bold
